# Quick Start Guide

## Prerequisites
- Node.js (v18 or higher)
- npm (comes with Node.js)

## Setup & Run

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open in browser:**
   - Navigate to `http://localhost:3000`

That's it! The app should now be running locally.

## Project Overview
A headcount planning model tool that calculates financial projections for startups based on team composition, revenue, and costs. Built with Next.js, React, TypeScript, and Tailwind CSS.
